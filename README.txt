Boss Dodge
====================

這是一個不需要額外安裝工具、可直接在瀏覽器打開遊玩的迷你閃避遊戲。

啟動方式：
1. 直接打開 index.html
2. 按「開始 / 重新開始」
3. 用鍵盤左右鍵或 A / D 操作

操作方式：
- 電腦：← → 或 A D
- 手機：在畫面上左右拖曳

專案結構：
- index.html   頁面主體
- css/style.css   樣式
- js/game.js   遊戲邏輯
- assets/   預留素材資料夾

備註：
- 舊的 mini-game.html 保留作為單檔 demo 版本
- 正式試玩請優先打開 index.html
